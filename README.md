# Envoix UI/UX Demo

Vue.js + TypeScript demo pages for Envoix transfer workflows.

## Routes

- `/demo/mobile` - Android-style mobile workflow, including Transfer, Receive, Send, and Settings tabs.
- `/demo/cli` - CLI workflow reference based on the current Envoix command surface.
- `/demo/gui` - Desktop GUI workflow mock for QR import, file selection, save-as, invite links, and short-token pairing.

## Development

```bash
npm install
npm run dev
```

Build and typecheck:

```bash
npm run build
```
